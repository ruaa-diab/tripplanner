# Ruaa Planner - Your Personal Travel Organizer

Ruaa Planner is an Android-based trip planning application designed to make travel preparation effortless and organized. It helps users create, manage, and track their travel tasks with an intuitive interface and beautiful design.

## Project Structure

### Key Activities:
- **MainActivity**: Home screen with task overview and filtering
- **AddTaskActivity**: Create new travel tasks with detailed information
- **EditTaskActivity**: Modify existing tasks and update details

### Core Features:
- **Task Management**: Create, edit, delete, and organize travel tasks
- **Smart Categorization**: Sort tasks by categories (Packing, Transport, Accommodation, etc.)
- **Priority System**: Color-coded priority levels (High/Medium/Low)
- **Date Planning**: Integrated date picker for task deadlines
- **Progress Tracking**: Mark tasks as completed and track your trip preparation progress

### Data Management:
- **Task Class**: Comprehensive task model with all travel-related attributes
- **SharedPreferences**: Local data persistence for seamless user experience
- **TaskAdapter**: Efficient RecyclerView adapter for smooth task listing
- **Gson Integration**: JSON serialization for reliable data storage

## User Experience

### Visual Design:
- **Color Scheme**: Soothing baby pink and purple theme
- **Material Design**: Modern Android components and smooth animations
- **Responsive Layout**: Adapts beautifully to different screen sizes

### Workflow:
1. **View All Tasks**: See your complete travel checklist at a glance
2. **Filter Views**: Switch between "To Do" and "Completed" tasks
3. **Add New Items**: Quick task creation with category and priority settings
4. **Track Progress**: Visual indicators show your preparation completion
5. **Stay Organized**: Never forget important travel preparations again

## Tech Stack

- **Development**: Android Studio
- **Language**: Java
- **Storage**: SharedPreferences with Gson serialization
- **UI Components**: RecyclerView, DatePicker, RadioButtons, CheckBox, Floating Action Button
- **Architecture**: Multi-activity with centralized data management
